Lunari Dictionary

Basic Rules

SVO (same as English)
過去形は語尾に「-na」をつける
例: mira → mirana
未来形は語尾に「-va」をつける
例: mira → mirava

冠詞は無し 三人称のsも無し 形容詞と副詞の区別も無いので、形容詞はそのまま副詞としても使える。 複数形もなくて前に数字をつけるだけ vaは未来 疑問系はke?を文末につけるだけ 性別の区別無し 主語 + (否定形) + 数 + 動詞 + ke? (疑問形) 形容副詞はどこにでもおける
  • I 私 mi
  • You あなた lu
  • like 好き sena
  • sky 空 aura
  • cute かわいい lili
  • go 行く naru
  • cat 猫 lumi
  • dog 犬 luma
  • sleep 寝る nemu
  • no いいえ nei
  • yes はい ya
  • want to したい mei

Example Sentences

I like the sky.
→ Mi sena aura.

I like you.
→ Lu sena lu.

The cute cat.
→ Lumi lili.
